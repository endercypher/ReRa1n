# ReRa1n
ReRa1n source code

How to use on linux:

git clone https://github.com/AidanGamzer/ReRa1n.git

cd ReRa1n

sudo cp rera1n.sh /usr/bin

sudo chmod +x /usr/bin/rera1n.sh

sudo apt-get update --fix-missing

ReRa1n.sh 

Leave all apt-get errors in the issues tab with the tag of [APT]
